# Custom Wallpapers

Put your CatalystUi wallpaper here before running the workflow.

Supported formats:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

The first image found in this folder is copied into the ROM as:
- `product/media/wallpaper/default_wallpaper_001.jpg`
- `product/media/wallpaper/default_wallpaper.jpg`
